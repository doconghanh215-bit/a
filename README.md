# 🎵 Full Stack Music Streaming Application

A modern, responsive, full-stack music streaming website similar to Spotify, YouTube Music, and Apple Music.

## 🛠 Technology Stack

### Frontend
- React.js + TypeScript
- Next.js 14+
- Tailwind CSS
- Redux Toolkit
- Socket.io-client
- PWA support

### Backend
- Node.js + Express.js
- PostgreSQL
- Redis
- Socket.io
- JWT Authentication
- AWS S3 / Local Storage
- Docker & Docker Compose
- Swagger API Documentation

## ✨ Features

### Core Features
- ✅ Authentication (Register, Login, OAuth)
- ✅ Music Upload & Management
- ✅ Advanced Playback Controls
- ✅ Playlist Management
- ✅ Search & Discovery
- ✅ Social Features
- ✅ Real-time Synchronization
- ✅ Offline Mode
- ✅ Admin Dashboard
- ✅ Analytics & Statistics

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/doconghanh215-bit/a.git
cd a

# With Docker
docker-compose up -d

# Access
- Frontend: http://localhost:3000
- Backend: http://localhost:3001
- API Docs: http://localhost:3001/api/docs
```
